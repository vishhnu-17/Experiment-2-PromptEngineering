# EXP-2-PROMPT-ENGINEERING-

## Aim: 
Comparative Analysis of different types of Prompting patterns and explain with Various Test Scenarios

Experiment:
Test and compare how different pattern models respond to various prompts (broad or unstructured) versus basic prompts (clearer and more refined) across multiple scenarios. 
Analyze the quality, accuracy, and depth of the generated responses.


## Algorithm:
## Scenario
Create a study plan for a college student preparing for semester exams while managing assignments and revision.
## 5 Prompting Techniques

### 1. Basic Prompting
Create a study plan for a college student.
### 2. Refined Prompting

Create a detailed 7-day study plan for a college student preparing for semester exams. Include study hours, revision, breaks, assignments, and sleep schedule.
### 3. Few-Shot Prompting

Example:

Day 1:

7–9 AM: Mathematics
10–12 PM: Programming

Day 2:

8–10 AM: Physics

Now create a 7-day study plan for a college student.
### 4. Chain-of-Thought Prompting
Create a study plan and explain step-by-step how you prioritized subjects, revision timing, and breaks.

### 5. Role-Based Prompting
Act as an academic productivity coach and create an effective study plan for a college student.
